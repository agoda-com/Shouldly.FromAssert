# Shouldly.FromAssert

Roslyn Code Fix library for migrating code from **NUnit** and **XUnit** assertions to **Shouldly Shoulds**.

### Supports the following list of assertions

- Assert.That
- Assert.True
- Assert.False
- Assert.Null
- Assert.NotNull
- Assert.IsEmpty
- Assert.IsNotEmpty
- Assert.AreEqual
- Assert.AreNotEqual
- Assert.AreSame
- Assert.AreNotSame
- Assert.Contains
- Assert.Zero
- Assert.NotZero
- Assert.IsNaN
- Assert.IsInstanceOf
- Assert.IsNotInstanceOf
- Assert.IsAssignableFrom
- Assert.Throws
- Assert.ThrowsAsync
- Assert.DoesNotThrow
- Assert.DoesNotThrowAsync
